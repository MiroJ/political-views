# Политически въпросник (Political Views Questionnaire)

Интерактивно уеб приложение за определяне на политически възгледи чрез въпросник.

## Описание

Това приложение представлява въпросник с 20 въпроса, които измерват политическите възгледи на потребителя по две оси:

1. **Икономическа ос (Ляво ↔ Дясно)** - 10 въпроса
2. **Социално-културна ос (Прогресивно ↔ Консервативно)** - 10 въпроса

> ⚠️ **Важна бележка:** Резултатите не са статистически представителни и не претендират за научна точност. Те са единствено ориентировъчни и служат само за информация и самопознание.

## Характеристики

- ✅ Начална страница с обяснение на въпросника и предупреждение за представителността на резултатите
- ✅ 20 въпроса с по 3 варианта за отговор всеки
- ✅ Визуален индикатор за напредък
- ✅ Детайлни резултати по двете оси
- ✅ Политически компас, показващ позицията на потребителя
- ✅ Цветно кодирани резултати
- ✅ Адаптивен дизайн

## Инсталация и стартиране

### Предпоставки
- Node.js (v18 или по-нова версия)
- npm
- Angular CLI

### Стъпки за стартиране

1. Отидете в директорията на приложението:
```bash
cd political-questionnaire
```

2. Инсталирайте зависимостите:
```bash
npm install
```

3. Стартирайте development server:
```bash
ng serve
```

4. Отворете браузър на адрес `http://localhost:4200`

## Как работи

1. **Начална страница**: Обяснение на въпросника и предупреждение за ненаучния характер на резултатите
2. **Въпросник**: Потребителят отговаря на 20 въпроса, избирайки един от трите варианта за всеки въпрос
3. **Точкуване**: Всеки отговор носи 0, 1 или 2 точки
4. **Изчисляване**:
   - Икономическа ос: Сума от точките на въпроси 1-10
   - Социално-културна ос: Сума от точките на въпроси 11-20
5. **Резултати**:
   - 0–33% = Ляво/Прогресивно
   - 34–66% = Център/Умерено
   - 67–100% = Дясно/Консервативно

## Данни

Въпросите са извлечени от Excel файла и конвертирани в JSON формат. Актуалните въпроси се намират в `political-questionnaire/public/questions.json`.

## Build

За production build:
```bash
ng build
```

Build артефактите ще се създадат в `dist/` директорията.

## Публикуване (Build and Publish)

Преди да започнете с публикуването, уверете се че имате `angular-cli-ghpages` NPM пакет инсталиран
на компютъра в глобалния NPM кеш. Ако го нямате, изпълнете

    npm install -g angular-cli-ghpages

За да публикувате този проект в GitHub Pages, изпълнете следните команди:

    cd .\political-questionnaire\
    ng build  --base-href https://github.com/MiroJ/political-views/
    npx angular-cli-ghpages --dir=dist/political-questionnaire/browser

След няколко секунди, апликацията трябва да е инсталирана на https://miroj.github.io/political-views/.

Успех!
